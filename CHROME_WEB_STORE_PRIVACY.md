# Chrome Web Store Privacy Summary

## Data Collection
Quantum Tab collects only the data you explicitly provide and configure:

- **Website visit counts**: Only for websites you choose to track via the Website Counter widget
- **API credentials**: GitHub PATs and Gemini API keys you enter for widget functionality

## Data Usage
- Website visit data is used only to display statistics in the widget
- API credentials are used only to access the respective third-party services (GitHub, Google Gemini)
- All data processing happens locally in your browser

## Data Storage
- All data is stored locally in Chrome's encrypted storage
- No data is transmitted to Quantum Tab servers or shared with third parties
- Data remains on your device only

## Data Deletion
- Clear website tracking data by removing websites from the counter
- Clear API credentials using the "Clear Credentials" button in widgets
- Uninstall the extension to remove all stored data

## Third-Party Services
- GitHub API (for pull request data)
- Google Gemini API (for AI background generation)
- Google Favicon Service (for website icons)

All third-party communications require your explicit API key/token configuration.

## Compliance
This extension complies with Chrome Web Store privacy requirements by collecting minimal data with user consent and providing clear data control mechanisms.</content>
