# Privacy Policy - Quantum Tab

**Last Updated: November 8, 2025**

## Overview

Quantum Tab is a Chrome extension that provides a customizable new tab experience with various productivity widgets. This privacy policy explains what data we collect, how we use it, and your rights regarding your data.

## Data Collection and Usage

### GitHub Integration (GitHub Guru Widget)

**What we collect:**
- GitHub Personal Access Tokens (PATs) that you provide
- Repository URLs you specify

**How we collect it:**
- PATs are entered by you in the widget configuration
- Repository URLs are entered by you in the widget configuration

**How we use it:**
- To fetch pull request data from GitHub's API
- To display repository information in the widget

**Data storage:**
- PATs are stored locally in your browser's Chrome storage
- No PAT data is transmitted except to GitHub's official API
- You can clear credentials using the "Clear Credentials" button

### AI Background Generation (Background Manager Widget)

**What we collect:**
- Gemini API keys that you provide

**How we collect it:**
- API keys are entered by you in the widget configuration

**How we use it:**
- To generate background images using Google's Gemini AI service
- API requests are made directly from your browser to Google's servers

**Data storage:**
- API keys are stored locally in your browser's Chrome storage
- No API key data is transmitted except to Google's official API
- You can clear API keys by removing or reconfiguring the widget

## Data Sharing and Transmission

**We do not:**
- Sell, rent, or share your personal data with third parties
- Transmit collected data to our servers
- Use collected data for advertising or marketing
- Track your browsing behavior across different websites

**Third-party services we use:**
- **GitHub API**: For pull request data (requires your PAT)
- **Google Gemini API**: For AI-generated backgrounds (requires your API key)
- **Google Favicon Service**: For website icons (public service, no authentication required)

All API communications are made directly from your browser to the respective service providers.

## Data Security

- All sensitive data (API keys, tokens) is stored locally in Chrome's encrypted storage
- No data is transmitted unless you explicitly configure third-party integrations
- You have full control over what data is collected and stored

## Data Retention and Deletion

- Website visit data is retained until you remove websites from tracking or clear the widget
- API keys and tokens are retained until you clear credentials or remove widgets
- You can delete all data by removing the extension or clearing browser data

## User Rights

You have the right to:
- View all data collected by the extension (stored locally in browser)
- Delete any collected data at any time
- Revoke permissions for any website tracking
- Clear all stored credentials and API keys
- Uninstall the extension to remove all data

## Changes to This Policy

We may update this privacy policy as needed. Any changes will be reflected in the extension's documentation and Chrome Web Store listing.

## Contact

If you have questions about this privacy policy or the extension's data practices, please check the extension's GitHub repository for support information.

## Chrome Web Store Compliance

This privacy policy is designed to be compatible with Chrome Web Store privacy requirements. The extension:
- Only collects data with explicit user consent
- Provides clear data deletion mechanisms
- Does not transmit data without user configuration
